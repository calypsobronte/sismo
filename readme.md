# Readme.md

Python 3

## Install virtualenv
```
pip install virtualenv
```

## crear el entorno virtual
```
virtualenv venv
```

## activar entorno virtual
```
source venv/bin/activate
```

## instalar requeriments
```
pip install -r requeriments.txt
```

## correr el servidor
```
python dummy.py
```
